# Roadmap (Index)
Last reviewed: 2026-06-04 UTC

This file is the global roadmap index for the current product direction.

## Active Direction
- Stable web/PWA is the first shipping target for non-Android users, especially iOS.
- Native Android is a separate offline-first product, not a wrapper around the web app.
- Demo web is the experimentation lane where features must mature before promotion into stable web/PWA and Android.
- Android distribution lane is Myket-first with optional dual-store publish path (`Myket + Cafe Bazaar`) when explicitly enabled per release.
- Distribution hub must remain metadata-aligned with controlled direct-download release artifacts (version/date/SHA-256 parity).
- Product scope lock is enforced by `npm run ops:scope:check`.
