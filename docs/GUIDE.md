# App Guide
Last reviewed: 2026-04-25 UTC

Guide summary for the current stable product.

## Quick Start
- Login or register.
- Choose a quick-game mode.
- Enter disposable player or team names.
- Set scoring rules and start playing.

## Core Controls
- Add score rounds.
- Undo the last round.
- Edit the most recent round.
- Continue unfinished local quick games after refresh when available.

## Profile And Settings
- Edit name, username, and email.
- Change password.
- Toggle sound and haptics.
- Change theme.

## Install And Update
- Android and desktop browsers may show a real install button.
- iOS Safari uses manual Add to Home Screen steps.
- Updates are checked manually and applied through the service worker lifecycle.
